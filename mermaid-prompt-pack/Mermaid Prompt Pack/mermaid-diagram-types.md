# Mermaid.js Diagram Types Reference

## Process & Flow Diagrams:
- **Flowchart**: Process flows, decision trees, workflows
- **User Journey**: Customer experience mapping, user interactions
- **State Diagram**: System states, transitions, conditions
- **Sequence Diagram**: Interactions over time, API calls, protocols

## Organizational & Structural:
- **Class Diagram**: Object relationships, software architecture
- **ER Diagram**: Database relationships, data modeling
- **C4 Diagram**: System architecture, component relationships
- **Block Diagram**: System components, high-level architecture

## Project & Time Management:
- **Gantt Chart**: Project timelines, task dependencies
- **Timeline**: Historical events, project milestones
- **Kanban**: Workflow states, task management

## Data Visualization:
- **Pie Chart**: Proportional data, market share
- **XY Chart**: Correlation data, performance metrics
- **Quadrant Chart**: Priority matrices, positioning maps
- **Sankey Diagram**: Flow quantities, process efficiency

## Specialized:
- **Git Graph**: Version control workflows, branching strategies
- **Mindmap**: Brainstorming, concept mapping
- **Requirement Diagram**: System requirements, dependencies
- **Packet Diagram**: Network protocols, data transmission
