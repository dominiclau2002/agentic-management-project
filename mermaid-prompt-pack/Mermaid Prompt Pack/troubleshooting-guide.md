# Mermaid.js Troubleshooting Guide

## Parse Errors:
- Remove special characters from node text
- Avoid quotes in classDef font names
- Simplify HTML formatting
- Check bracket matching

## Styling Issues:
- Use unquoted font names
- Ensure hex colors include #
- Avoid complex CSS properties
- Test syntax in Mermaid Live

## Export Problems:
- Optimize font sizes for target format
- Use high contrast colors
- Avoid tiny text elements
- Test export before finalizing
